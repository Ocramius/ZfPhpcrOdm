ZfPhpcrOdm - Zend Framework 2 module for Doctrine PHPCR ODM

This Module provides functionality to connect Zend Framework 2 ( http://packages.zendframework.com/ ) and Doctrine PHPCR ODM ( https://github.com/doctrine/phpcr-odm ). The module provides configuration and dependencies for Jackrabbit, Doctrine DBAL DBAL and Midgard 2 JCR backends.

Installation

$ cd path/to/your/zf/application

Then create a composer.json file like following:

{
    "require": {
        "ocramius/ZfPhpcrOdm": "dev-master"
    }
}

Then run composer installation:

$ curl -s http://getcomposer.org/installer | php
$ php composer.phar install

Then add ZfPhpcrOdm to your loaded modules and vendor/ocramius to your module_paths in config/application.config.php.

Done! You can now use a Doctrine\ODM\PHPCR\DocumentManager instance in your project. Don't forget to read the docs about PHPCR ODM!
